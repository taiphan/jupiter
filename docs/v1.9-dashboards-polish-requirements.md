# v1.9 — Dashboards, polish & exports

**Status:** Shipped (as release **1.13.0**)  
**References:** [REQUIREMENTS.md](../REQUIREMENTS.md)

---

## Goals

1. **Customizable project dashboards** — velocity, burndown, cumulative flow widgets per project.
2. **Enhanced For You** — due-soon and what-changed sections on My Work.
3. **CSV export** — issues list and velocity report.
4. **Issue email hooks** — optional SMTP notifications on status/assignee/priority/comment events.

---

## Deliverables

| Area | Implementation |
|------|----------------|
| Dashboard | `/projects/{key}/dashboard` + `dashboard-store` widget toggles |
| For You | `derive/for-you.ts` — due soon + recent activity |
| CSV | `lib/export/csv.ts` — issues + velocity |
| Email | `POST /api/notify/issue-event` when `ISSUE_EMAIL_NOTIFICATIONS=true` |

---

## Ops

```bash
# Optional: email watchers/assignee on issue events (uses existing SMTP)
ISSUE_EMAIL_NOTIFICATIONS=true
```

Requires Gmail/SMTP configured (v1.10).
